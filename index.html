About Me

I am a Spring developer with experience in building and managing applications. Here you can see some of the projects I have worked on.

Projects

Project 1: Employee Task Manager

This project is a Spring Boot application integrated with a MySQL database. The application manages two main entities: Employee and Task, with a one-to-many relationship between them. Additionally, it handles the authentication and authorization process using a User and Role relationship, implemented as a many-to-many association. The presentation layer is built using Thymeleaf, facilitating server-side data rendering.

Project 2: Bookstore Application

Developed a responsive and user-friendly bookstore web application using Spring Boot and Thymeleaf. Features include secure user registration and login, a dynamic book catalog with purchase options, and a responsive design optimized for both mobile and desktop views.
