About Me

I'm a software engineer (backend Focus) passionate about building secure, scalable, and well-architected systems using Java, Spring Boot, and Spring Framework. I design clean REST APIs, implement JWT-based RBAC authentication (with Auth0), and structure code for long-term maintainability. My skill set includes SQL, Docker, Swagger, and a strong focus on data validation, exception handling, and automated testing (unit and integration). I also leverage GitHub Actions to automate CI workflows, ensuring code quality and continuous delivery. I'm always striving to write clean, reliable code that supports real-world business logic and grows with the product.
